Who is front-end Developer?

lorem22

Front -end developer istifadəçilərə sayt və ya proqrama daxil olmaq və onlarla qarşılıqlı əlaqədə olmaq imkanı verən HTML, CSS və JavaScript kimi veb dillərindən istifadə edərək veb-saytlar və proqramlar yaradır.Front-end tərtibatçıları istifadəçi interfeysləri (UI) yaradırlar. UI saytın və ya tətbiqin hər bir hissəsinin nə etdiyini və necə görünəcəyini müəyyən edən proqramın qrafik tərtibatıdır. Front-End İnkişafının əsas dilləri bunlardır: HTML, CSS, JavaScript.
